Per the HTML Living Standard, The dialog element represents a part of an application that a user interacts with to perform a task, for example a dialog box, inspector, or window.

A modal dialog is a window or overlay that requires the user's immediate attention, blocking any interaction with the underlying content until the dialog is closed.

Non-modal dialog does not restrict user interaction with the underlying content. It allows users to continue interacting with other elements on the web page while the dialog is open.
